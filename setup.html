Configuración Online - Supabase

De: Project Settings → API → Project URL
De: Project Settings → API → anon (public)
📝 Cómo obtener credenciales:

1. Ve a supabase.com
2. Abre tu proyecto
3. Click en Settings (rueda abajo)
4. Vé a API
5. Copia Project URL
6. Copia anon (public) key
7. Pégalas aquí

⚠️ Importante: Copia las credenciales tal cual, sin espacios.
